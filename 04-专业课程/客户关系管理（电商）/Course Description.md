# 课程简介

- 客户关系管理是一门将先进的管理理念、创新的管理机制与现代信息技术相结合的新兴综合性交叉学科，是市场营销、电子商务等经济管理类专业的重要课程。
- 本课程既注重理论的学习，同时又非常强调实践应用，研究内容具有**综合性、实践性和应用性**等特点。

# 学习目标 → 课程体系

## 学习目标 (What, Why, How)

- **what**：什么是“以客户为中心”？
- **why**：为什么要“以客户为中心”？
- **how**：如何实现“以客户为中心”？

## 核心问题

### 课程体系核心问题：
- 谁是企业的客户？
- 客户于企业而言有何重要价值？
- 以客户为中心的思想是如何产生和发展的？

### 课程重点解决的核心问题：
1. **科学评估客户价值**  
   基于客户价值实现客户分类管理是客户关系管理的基础工作。——需要解决客户信息收集和选择哪种方法评估客户价值等问题。

2. **客户满意与客户忠诚**  
   是“以客户为中心”的目标，是维系老客户的必由之路。——需要解决如何实现客户满意和建立客户忠诚等问题。

3. **客户关系管理信息技术应用**  
   ——解决管理效率问题。
