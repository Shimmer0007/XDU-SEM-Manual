# 考核方式

| 课程目标   | 平时作业 | 综合性大作业 | 课堂表现 | 期末考试 |
|------------|----------|--------------|----------|----------|
| 课程目标1  | 5        |              |          | 20       |
| 课程目标2  | 5        |              |          | 15       |
| 课程目标3  | 5        |              |          | 15       |
| 课程目标4  |          | 10           | 5        | 20       |
| **合计**   | **15**   | **10**       | **5**    | **70**   |

本课程的考核以考核学生能力培养目标的达成为主要目的，以检查学生对各知识点的掌握程度以及应用为重要内容。最终成绩由平时作业成绩和期末成绩组合而成，各部分所占比例如下：

1.  **平时成绩：30%**。分为3次平时作业、综合性大作业和课堂表现等。
2.  **期末成绩：70%**。对学习情况进行全面的检查，考试基本理论知识和综合应用题。
