# 建站作业评分标准

| 分类       | 子项               | 具体项目                                                     | 分值 |
|------------|--------------------|--------------------------------------------------------------|------|
| 页面       | 布局               | 结构新颖、有创意、规范                                       | 4    |
|            | 美观性             | 1. 文字清晰，标题主次分明                                     | 4    |
|            |                    | 2. 背景恰当，突显文字，美观                                   | 4    |
|            |                    | 3. 页面布局有条理                                             | 4    |
|            |                    | 4. 色彩使用合理，不会因为过度使用颜色使页面变得杂乱           | 4    |
|            |                    | 5. 整站主题色同意且与网站整体主题契合，保持统一风格           | 4    |
|            | 主题               | 1. 各个页面均能表达网站整体主题                               | 4    |
|            |                    | 2. 各个页面主题突出                                           | 4    |
|            | 其他               | 页面内容不超过3帧（屏）、不少于1帧（屏）                     | 4    |
| 单页面数量超过4页 |                    | 1. 页面没有完成或未进行排版，该页不计在内                     | 4    |
|            |                    | 2. 层次清楚合理                                               | 4    |
| 层次       | 链接、导航         | 1. 链接正确性                                                 | 3    |
|            |                    | 2. 首页可链接到任意下级分页                                   | 3    |
|            |                    | 3. 各个页面可直接返回首页                                     | 3    |
|            |                    | 4. 锚文本、按钮等在鼠标悬停时有明显的链接提示                 | 3    |
|            |                    | 5. 链接样式合适，不影响网页整体美观                           | 3    |
|            |                    | 6. 导航简单、清晰，不超过3个层级                             | 3    |
|            |                    | 7. 列表页面（如：产品中心、新闻中心）有上一页、下一页按钮     | 3    |
| 效果       | 图片处理           | 1. 图片显示正常（清晰、完整）                                 | 3    |
|            |                    | 2. 图片选择恰当，符合主题                                     | 3    |
|            |                    | 3. 自制banner图片，根据美观度给与分数                         | 3    |
| 技术       | 模板、框架动态特效等技术 | 1. 网站布局结构层次分明，上下级导航有较强关联性               | 3    |
|            |                    | 2. 动画效果使页面重点突出，无杂乱感                           | 3    |
| 内容       |                    | 1. 遵守国家法律（如：计算机信息网络国际联网安全保护管理办法、新广告法等） | 4    |
|            |                    | 2. 文字内容层次清晰，排版合理，符合阅读习惯                   | 4    |
|            |                    | 3. 新闻、产品各做2个以上分类                                 | 4    |
|            |                    | 4. 每个分类上至少2篇内容                                     | 4    |
|            |                    | 5. 列表中的每篇内容不少于300字                               | 4    |
